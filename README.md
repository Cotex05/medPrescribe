# medPrescribe
 ## An expo based React Native App.
 
### To run this project:
* First Clone it - `git clone https://github.com/Cotex05/medPrescribe.git`
* Go to project directory.
* Open command line and run `npm install` to install the required depedencies.
* Then run `expo start`.
